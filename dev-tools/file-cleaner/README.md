# DWNFDCleaner

**Interactive File Deletion Utility**

A simple CLI tool for safely deleting individual files from the filesystem with confirmation prompts.

## Usage

```bash
python main.py
```

When prompted, enter the full path to the file you want to delete:

```
Insert the path of the file to be deleted: /home/user/Downloads/file.pdf
```
