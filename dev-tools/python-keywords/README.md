# Python_kws

**Python Keywords Reference Tool**

A quick utility to display all Python reserved keywords and soft keywords, useful for developers learning Python or checking syntax.

## Usage

```bash
python py_kws.py
```

## Output

Displays a numbered list of:
- **Keywords**: Reserved words (`if`, `for`, `class`, `def`, etc.)
- **Soft Keywords**: Context-dependent keywords (`match`, `case`, `_`)

## Example Output

```
Keywords:
 1: False
 2: None
 3: True
 4: and
...

Soft keywords:
 1: _
 2: case
 3: match
```

## Requirements

- Python 3.10+ (for `softkwlist` support)
