# Youtubedownloader

**YouTube Video Downloader**

A lightweight CLI wrapper for downloading YouTube videos at the highest available resolution.

## Usage

```bash
python main.py
```

When prompted, paste the YouTube video URL:

```
Insert the youtube video URL here: https://www.youtube.com/watch?v=VIDEO_ID
```
